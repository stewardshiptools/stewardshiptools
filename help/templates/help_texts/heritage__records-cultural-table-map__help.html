  • listCultural Records Table and Map

The Cultural Records table and map show all cultural site/story records collected in the Marine TEK interviews.

Table Notes
  • The table/map may be PRE-filtered by Interview, IF this table/map is found on an Interview Details page. If this table/map is on the main Cultural Records page, you will see ALL records.
  • The table and map are LINKED. As you filter the table, the map will automatically reload.
  • Results per page - change the number of records show in the table/map using this drop-down field. The limit is 1000 records per page.
  • Filter... - filter the table (and thus the map) by entering text strings. The following MTK fields have been indexed: ID, Use Description, Species Description, Time Frame Start Description, Time Frame End Description, Harvest Method Name, Fishing Method Description, Ecological Value Description, Species Theme Name, Temporal Trend Description, and Link Code.

Map Notes
  • Zoom - click the +/- buttons at the top-left of the map OR use your scroll wheel OR pin gesture on a tablet or touch screen
  • Pan - drag you cursor or finger/stylus on a touch screen
  • Show in Full Screen - click to see the map expand to show in your full browser window
  • Base Map - Use the radio buttons in the Layer Legend, top-right of map, to choose a base map like Open Street Map (OSM) Mapquest or Mapquest Satellite
  • Overlay Layer(s) - turn on/off the overlay layers using the checkmark buttons below the Basemap layers. Overlay layers may have a Pencil button and a Garbage-can button: use these to edit or delete the overlay layer.

Examples
  • Table - try changing the number of results from 10 (default) to 500.
  • Table - try entering “Village 32” into the Filter field - two results are returned; both village sites, one with ID =132 and the other with spatial code E032.
  • Try clicking on a feature (point, line or polygon) on the map. The pop-up will show you a name and a link to the specific record.
