Instructions
You can now make changes to these fields:
  • Name - Change the name. It is recommended that you include the Project or Heritage Interview ID in the name.
  • Location - The Input Type defaults to the method originally used to create the Spatial Layer. Update the Location using the method-specific instructions below.
  • Notes - Edit Notes text

Can I change the Input type?
Yes. However changing the Input type will delete any features created using the previous/original input.

Update Location using using revised WKT
Follow these steps:
  1. Copy and paste the updated WKT text into the "Wkt" field.
  2. Edit Notes field as needed.
  3. Click the UPDATE PROJECT LOCATION button to save your changes.

Update Location using using Draw on a map
Follow these steps:
  1. Activate the pencil button to create one or more point features. Each single-mouse-click with the pencil button activated will add a point feature.
  2. Activate the line button to create one or more line features. Start a line with a single-mouse-click and add vertices (turns) with another single-mouse-click. End the line drawing with a double-mouse-click.
  3. Multiple line features are allowed.
  4. Activate the polygon button to create one or more polygons. Start a polygon with a single-mouse-click and add vertices (turns) with another single-mouse-click. End the polygon drawing with a double-mouse-click.
  5. It is a good habit to deactivate your tool by clicking the hand tool.
  6. Edit your features by clicking on the edit button. Point: select and move. Line: select, then move a vertice or add a vertice by dragging an intermediate vertice. Polygon: select and move any vertices, and/or create a new vertices by dragging an intermediate vertices.
  7. Delete features using the Delete all features link below the map.
  8. Click the UPDATE PROJECT LOCATION button to save your changes.

Update Location using a replacement Shapefile
Follow these steps:
  1. Preparation: Shapefiles must be a zipped collection of all the individual Shapefile files. It is recommended that the name of the zip file be the same as the contained files. E.g. "project_polygon.zip" would include the following files: project_polygon.shp, and project_polygon.dbf and project_polygon.prj. Multi-part polygons are NOT allowed.
  2. (Optional) Preparation: Use ArcGIS or QGIS to examine the dbf file and determine what attribute labels are available. Make note of these labels and think about the titles you would like for individual features in the map layer.
  3. (Optional) Designate an automatically generated feature label, e.g. My layer feature %ID% %poly name%. Note that feature labels are not available from KMZ files.
  4. Select File and add (a) Title and (b) Notes as instructed above above.
  5. Click the FILE button and locate the zipped Shapefile on your local system.
  6. Click the UPDATE PROJECT LOCATION button to save your changes.

Update Location using a replacement KMZ (zipped Google KML file)
Follow these steps:
  1. Preparation: Save/export your layer(s) in Google Earth as a KMZ. Warning: don't make this too complex! Also note that feature attributes are not always ingested from a KMZ.
  2. Select File and add (a) Title and (b) Notes as instructed above above.
  3. Click the FILE button and locate the GMZ file on your local system.
  4. Click the UPDATE PROJECT LOCATION button to save your changes.
