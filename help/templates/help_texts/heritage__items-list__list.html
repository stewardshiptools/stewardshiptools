{% load static %} A page listing all the Heritage Items currently in the database that you can filter and sort. The page has the following parts: Quick Filter, (right-side) Reset, Sort, Advanced Filters and Add New. You can also use the full Heritage Search page to find content, including excerpts of text near search results.

searchQuick Filter



Use the Quick Filter to quickly find a subset of Items. Quick Filter filters on:
  • ID, with or without the prefix - e.g. try typing "I-1" or "1"
  • Name, including part of a name - e.g. try typing "I" or "Island"
  • Cataloger
  • Reviewer
  • Type
  • Creation Date
restoreReset


To reset the table, click the RESET FILTERS button, or delete all terms in the Quick Filter field and press [Enter] on your keyboard.

sortSort

Change the number of result rows by changing the Result per page value. The default is 25 results/rowsper page.

Sort the table by chosing a sort field (e.g. ID, Name) and then choose ascending or descending. The default table sort is by ID, descending.

filter_listAdvanced Filters (right side of table)
Try filtering the table by selecting values for:
  • Creation Date - after
  • Creation Date - before
  • Item Types
  • Collections
  • People Mentioned
  • Plants
  • Animals/li>
  • Spreadsheet ID/li>
Add New
Click the add_circle ADD ITEM button top-right to start cataloging a new Item.
