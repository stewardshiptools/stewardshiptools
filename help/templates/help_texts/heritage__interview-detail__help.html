Each Interview Details page has (left) an Information column, (right) an Assets List table on the right, (below) a Species Records linked table and map, and a Cultural Records linked table and map.

Information column

  • Staff - names, roles and links;
  • Location - where the interview sessions were held;
  • Dates - dates of sessions; and
  • Statistics - gross numbers of records collected and assets appended

Assets List table
The Assets List has these columns:
  • Name - name of asset (document) with a link for downloading. Warning: these files may be large.
  • Session - which interview session the asset relates to;
  • Type - Map, Audio, etc.; and
  • Size - in MB. This should give you an indication as to whether you should download it.

How to Filter the Assets table
Enter part of a Name or Type and the table will filter as you type. For example, typing Map will filter to table to only show Maps and Georeferenced Maps.
