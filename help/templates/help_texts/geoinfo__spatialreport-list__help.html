• listSpatial Reports Page Instructions

This page lists all the Spatial Reports currently in the system, 20 to a page.

Exploring the Table
Use the pagination tool to move between pages.
Click on the Report's name to view details of the report. Note: this also runs the report.
Filter the table by typing in the Filter... field search. Here is what the table filters on:
  • Report name, including part of a name
  • Default distance cap, including part of term
  • Report on item, including part of a name.
How to reset the filters To reset the table, delete all terms in the Filter field and press Enter on your keyboard, or reload the page using the browser reload button refresh.

Examples
Try exploring the table by:
  • Names: Try entering an report name, or part of a name e.g. 'Spec' to the Filter field will get all reports with 'species' or 'special' in the name.
  • Distance cap: 'pro' Try entering a value, e.g. '5'. Note that this si a text filter not a distance filter. You cannot enter '>500m', for example.
  • Report on item: try entering the filter term 'pro'; it will show all Spatial Reports with 'project' as a Report on item. This is a handing way to see all reports used to assess Development Projects.
How to add a new Spatial Report
Click the add_circle ADD SPATIAL REPORT button top-right to start creating a new Spatial Report.
