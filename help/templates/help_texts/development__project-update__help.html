Edit a Development Project using this form.

Instructions
Update as many or as few fields as you need to track the project. Note that you will add or edit Spatial Location data in the LOCATION tab.

The mandatory fields are:
  • Cedar Project name
  • Cedar project code
  • Consultation stage
Recommendations
    • Name - recommend uses a standardized format that clearly summarizes the project: APPROX-LOCATION - COMPANY - PERMIT-TYPE - DATE. Abbreviate location and company names and use a date format of YEARMONTHDAY. See example below.
    • Tags - any phrase that adds this project to a group of projects
    Examples
      • Name example: "Deep Bay - ABC Forest - LOO x2 Log Drops - 20160418"
      • Tag examples - "forestry" or "archaeology"
