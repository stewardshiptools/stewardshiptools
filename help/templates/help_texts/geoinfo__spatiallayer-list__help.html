• listSpatial Layers Page Instructions

This page shows all the GeoInfo layers in the database.
*There are app-specific GeoInfo Layer lists in the Heritage, Development and Ecosystems Apps (in prep).

About GIS Layers
Spatial features are generated in a few differently places in the Apps:
  • Created here in the GeoInfo Tools, using the ADD LAYER button.
  • Created in the Development App as a Location for a Development Project.
  • Programmatically, from other data. For example, the HMTK Species and HMTK Culture layers are custom one-time translations of the HMTK database tables (done in May 2016). Newer data added to HMTK Species and HMTK Culure tables through the Heritage App will not be in the GeoInfo layers until another translation is made.

Table exploration instructions
The Table shows layer names, number of features, authors and creation dates, and descriptions. Note how useful a good layer description is! 20 layers are shown to a page. Use the pagination to move between pages.
Filter the table by typing in the Filter... field search. Here is what the table filters on:
  • Layer name, including part of a name
  • Description, including part of a description
How to reset the filters
To reset the table, delete all terms in the Filter field and press Enter on your keyboard, or reload the page using the browser reload button refresh.

Examples
Try exploring the table by:
  • Names: Try entering an Layer name, or part of a name (e.g. 'Ha') to the filter field - results will be all reports with 'Haanas' and 'Haida' in the name.
How to add a new Layer
Click the add_circle ADD LAYER button top-right to start creating a new Layer.
