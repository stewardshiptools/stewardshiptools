This page lists all the Persons currently in the Contacts database. Persons are shown 20 to a page.

Table instructions
Use the pagination tool to move between pages.
Click on the Contact's name to view details.
Filter the table by typing in the Filter... field search. Here is what the table filters on:
  • Contact's first and /or last name, including part of a name
  • Contact's organization
  • Contact's role / job title
  • Indigenous Names
How to reset the filters/table To reset the table, delete all terms in the Filter field and press Enter on your keyboard, or reload the page using the browser reload button refresh.

Examples
Try exploring the table by:
  • Entering a job title e.g. 'interviewer' to the Filter field - to see all interviewer contacts
  • Entering part of a term, e.g. 'da' - returns several results
  • Entering an Indigenous Name, e.g. 'Jingwas'
  • Entering an organization name (or part of such term)
