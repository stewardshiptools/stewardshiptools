This page lists all the Heritage Projects currently in the system. Projects are shown as cards, 10 to a page.

Filter instructions
Use the pagination tool to move between pages.
Filter the table by typing in the Filter... field search. Here is what the table filters on:
  • Project name, including part of a name
How to reset the filters To reset the table, delete all terms in the Filter field and press Enter on your keyboard, or reload the page using the browser reload button refresh.

Examples
Try exploring the table by:
  • Names: Try entering an Project name, or part of a name (e.g. 'Pha') to the filter field - results will be all reports with 'phase' and 'phabulous' in the name.
How to add a new Project
Click the add_circle ADD PROJECT button top-right to start cataloging a new project.
