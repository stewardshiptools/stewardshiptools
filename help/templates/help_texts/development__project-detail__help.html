This page lists all the information for one Development Project.

Tab instructions
Click on the tabs to view additional information:
  • MAIN - People, dates and description
  • LOCATION - an interactive map and spatial upload tools (see notes below on exploring the map)
  • FILES - Any uploaded fiels describing the Project

Instructions for editing a project's general information
Follow these steps:
  1. Click on the EDIT PROJECT button and edit out the form.
  2. Click UPDATE PROJECT button.

Instructions for editing a project's spatial location
Follow these steps:
  1. Click on the the LOCATION tab.
  2. Click on the add_location button in the map.
  3. Fill out the form (see instructions on that page).
  4. Use the mode_edit and delete buttons to edit (e.g. re-upload/draw) or delete a spatial layer for this Project's location..
  5. "Drill-down" - Click on the drill-down play_for_work button and click on overlapping spatial layers to get more information.

Instructions for adding a file to a project
Follow these steps:
  1. Click on the ADD FILE button and fill out the form.
  2. Click the green FILE button to select a file from your computer.
  3. Choose what type of file from the Asset type list.
  4. Add a Comment. The is especially useful to note when, who and where the file is from.
  5. Click ADD PROJECT FILE button.
