This page lists all the information for one Heritage Project.
Each Heritage Details page has an Information panel on the left side, and then tabbed data on the right.

Information panel (left)
The panel contains background information that you can be edited by clicking the EDIT PROJECT button.
  • Project thumbnail - An image emblematic of the project
  • Project code - An optional code for the project
  • Project location(s) - Where the heritage interviews or surveys took place
  • Tags - Quick project classifications
  • Dates - The start and end date of all the interviews
  • Background - A brief summary of the project

Tabs (middle/right)
Click on the tabs to view additional information:
  • PEOPLE - People involved in the project, arranged as cards with thumbnail images
  • INTERVIEWS (default) - A table showing interviews, interviewers codes and interview numbers
  • FILES - Any uploaded files describing the Project

Instructions for editing a project's general information
Follow these steps:
  1. Click on the EDIT PROJECT button.
  2. Edit the form.
  3. Click UPDATE PROJECT button.

Instructions for adding a file to a project
Follow these steps:
  1. Click on the ADD FILE button and fill out the form.
  2. Click the green FILE button to select a file from your computer.
  3. Choose what type of file from the Asset type list.
  4. Add a Comment. The is especially useful to note when, who and where the file is from.
  5. Click ADD PROJECT FILE button.
