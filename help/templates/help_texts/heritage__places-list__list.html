{% load static %} A page listing all the Named Places currently in the database that you can filter and sort. The page has the following parts: Quick Filter, (right-side) Reset, Sort, Advanced Filters and Add New. You can also use the full Heritage Search page to find content, including excerpts of text near search results.

NOTE! The LIST and MAP tabs are linked. If the LIST shows only 25 Places, then the MAP will only show 25 Places. To see all Places on the MAP tab, select All from the 'Results per page', as explained below.

searchQuick Filter



Use the Quick Filter to quickly find a subset of Places. Quick Filter filters on:
  • ID, with or without the prefix - e.g. try typing "CB-1" or "1"
  • Story Title, including part of a name - e.g. try typing "I" or "Island"
restoreReset


To reset the table, click the RESET FILTERS button, or delete all terms in the Quick Filter field and press [Enter] on your keyboard.

sortSort

Change the number of result rows by changing the Result per page value. The default is 25 results/rows per page.

Sort the table by chosing a sort field (e.g. ID, Name) and then choose ascending or descending. The default table sort is by ID, descending.

filter_listAdvanced Filters (right side of table)
Try filtering the table by selecting values for:
  • Keywords
  • Tags
