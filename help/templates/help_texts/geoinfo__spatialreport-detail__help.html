Notes
codeThe Report query script starts as soon as the page loads. Note the moving band of colour at the top of the page. This means that the query is actively comparing the Report On and the Report Against layers' features.
codeCalculating proximity hits and distances is an intensive task for the server, especially if the layers are rich with features. Be patient.
codeYou can interupt the query at any time - it will not damage the database. Interupt the query script by changing one of the layers (items) or by clicking DISMISS on the bottom-left of the page.

Explore the Report Page
Each Report page has (1) a header, (2) a layers column on the left side, and (3) a main Results section on the right.

1. Header parts
  • Report name
  • Results statistic (number of overlapping or proximal features)
  • An mode_editEDIT button - click to change the Report name.
  • A deleteDELETE button - click to delete this Report.
  • A help button.

2. Layers Column parts
  • Report name
  • Results statistic (number of overlapping or proximal features)
  • Report on layer (item) - Change the layer by clicking on the mode_edit pencil icon. You could, for example, change its distance cap.
  • Report against layers (items) - Add another layer by clicking on the blue + button; Change a layer by clicking on the mode_edit pencil icon
    • ;Remove a layer from the lsit of Report on layers by clicking on the delete garbage can icon.

3. Main Results Report parts
  • LIST tab - provides a list of all 'hits'; Report aginst features that are intersecting and/or proximal to the Report on target layer's feature(s).
  • MAP tab - shows a map of the Report on (taget layer features and any Report against layer features that meet the distance cap rule. The map usually loads faster as distance calculations do not have to be completed.

3.b. How to use the Slippy Map
Try these:
  • Map feature(s) (red)- the Report on layer features. Click for feature info (pop-up).
  • Map feature(s) (blue) - the 'hits', the results, features that are proximal or overlapping. Click for feature info (pop-up).
  • Zoom - click the +/- buttons at the top-left of the map OR use your scroll wheel OR pin gesture on a tablet or touch screen
  • Pan - drag you cursor or finger/stylus on a touch screen
  • Show in Full Screen - click to see the map expand to show in your full browser window
  • Base Map - Use the radio buttons in the Layer Legend, top-right of map, to choose a base map like Open Street Map (OSM) Mapquest or Mapquest Satellite
  • Overlay Layer(s) - turn on/off the overlay layers using the checkmark buttons below the Basemap layers.
